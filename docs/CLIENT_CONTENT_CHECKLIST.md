# Client Content Collection Checklist

## 1. Brand Identity

Collect the following brand identity assets from the client before any page content work starts.

### Branding Assets

- Logo files in SVG and PNG formats.
- Header logo current asset size: 389x100 px (`logo-dark.png`).
- Footer logo current asset size: 389x100 px (`logo-light.png`).
- Favicon current asset size: 512x512 px (`favicon.png`).
- Recommended exported favicon variants for production: 32x32 and 180x180.
- Brand color palette:
  - Primary color (HEX)
  - Secondary color (HEX)
  - Accent color (optional)
- Brand fonts (if custom), including license-safe files and usage guidance.

### Company Profile Data

- Official company name.
- Phone number.
- Email address.
- Physical address.
- Working hours.

### Social Media Links

- Facebook URL.
- Instagram URL.
- Twitter/X URL.
- LinkedIn URL.

### Delivery Format Requested From Client

Ask the client to submit these files and values in a folder named `01-brand-identity` using this structure:

- `logos/` (SVG and PNG)
- `favicon/` (source 512x512 plus generated 32x32 and 180x180)
- `colors-fonts/` (brand colors, font files or references)
- `company-info.txt` (name, phone, email, address, working hours)
- `social-links.txt` (all profile URLs)

### Acceptance Checklist

- Logos are high quality and transparent where needed.
- Colors are provided as HEX values.
- Font usage rights are confirmed.
- Contact information is final and approved.
- All social links open correctly.
